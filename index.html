Jira API cho AI

Webhook events (tự cập nhật 3s/lần). Sửa một ticket trong Jira để thấy event mới.
Bấm một event bên trái hoặc nhập issue key ở trên để xem AI context — đúng JSON mà AI sẽ nhận, kèm trace các API đã gọi.
Đang gọi chuỗi API Jira...
Toàn bộ API service sẽ dùng, nhóm theo vai trò trong flow: webhook trigger → đọc hiểu ticket → tìm ticket tương tự → phán đoán & hành động → danh mục ngữ cảnh. Mỗi API ghi rõ nó giúp gì cho AI. Bấm Gọi thử để thấy response thật từ Jira (placeholder {key}/{project} lấy từ ô issue key ở góc phải trên). Chỉ API GET mới gọi thử được — API ghi/sửa hiển thị cách dùng.
